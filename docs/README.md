# Documentation authority map

All current documents in this directory are canon: they describe the behavior shipped by
the current package version.

- [Quickstart](quickstart.md) — smallest working setup for sync, async, persistent, and structured caching
- [API reference](api.md) — public interfaces, validation, key types, modes, stores, and telemetry
- [Design decisions](design.md) — why the API uses explicit config, structured opt-in, and service-owned single-flight
- [Eval loops](eval-loops.md) — production pattern for iterative AI-pipeline evaluation

There are currently no intent documents (PRDs, research snapshots, or roadmap) stored in
the repository. The implementation spec for the 0.3.0 hardening wave lives in the GitHub
issue tracker.
